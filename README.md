# ai_project_s28898
Repozytorium zawiera kod źródłowy projektu zaliczeniowego kursu ASI na PJATK.

## Temat projektu
Efektem końcowym ma być aplikacja klasyfikująca gatunki książek na podstawie tagów przypisanych przez użytkowników portalu `goodreads`. Celem jest ujednolicenie zbioru, utworzenie klasyfikacji wolnej od niespójności (literówki, zdrobnienia nazw gatunków itp.) oraz dopasowanej do realnej aktywności użytkowników, a nie arbitralnej oceny moderatora. To nieodzowne przedsięwzięcie, by powstały zbiór mógł służyć jako baza dla systemu rekomendacji czy punkt wyjścia głębszych analiz.
